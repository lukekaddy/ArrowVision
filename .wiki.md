# Project Summary
**ArrowLive** is a mobile-first web application designed for managing and scoring archery tournaments. It replaces traditional paper scorecards with a digital platform, providing features like live scoring, real-time leaderboards, and instant results. This innovative solution enhances the experience for tournament organizers, archers, and scorekeepers, ensuring efficient management of archery events.

# Project Module Description
- **Authentication**: User authentication via Atoms Cloud.
- **Tournament Management**: Tools for organizers to create, edit, and manage tournaments, including naming courses and specifying target counts.
- **Live Scoring**: Real-time input and updates of scores.
- **Leaderboard**: Dynamic leaderboard showcasing current rankings, with course filtering capabilities.
- **Smart Score**: AI-assisted scoring feature utilizing image capture technology.
- **Results Export**: Functionality to export tournament results in JSON format.

# File Description Inventory
- **backend/**: Contains the backend logic, including models, services, and API routers.
- **frontend/**: Contains the frontend application built with React, including UI components and pages.
- **README.md**: Documentation for both frontend and backend modules.
- **requirements.txt**: Lists dependencies for both frontend and backend.

# Technology Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS, shadcn/ui
- **Backend**: Atoms Cloud (FastAPI, PostgreSQL, Authentication)
- **SDK**: @metagptx/web-sdk for authentication and API calls

# Usage
1. **Install Dependencies**:
   - For the backend: Run `pip install -r requirements.txt`
   - For the frontend: Run `pnpm install`
   
2. **Build the Project**:
   - For the backend, ensure the database is set up and migrations are applied.
   - For the frontend, run `pnpm run build` to compile the application.

3. **Run the Application**:
   - Start the backend server.
   - Start the frontend server to access the web application.
