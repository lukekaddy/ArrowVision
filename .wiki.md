# Project Summary
**BullsEye Labs** is a mobile-first web application that revolutionizes archery tournament management through digital scorekeeping. It provides tournament organizers, archers, and scorekeepers with features like live scoring, real-time leaderboards, and instant results. The platform supports various tournament formats and advanced scoring options, enhancing user experience with tools such as the "Replay Camera," which allows users to capture and upload video replays of arrow impacts.

# Project Module Description
- **Authentication**: User authentication via Atoms Cloud with role-based access.
- **Tournament Management**: Tools for creating, editing, and managing tournaments, including course naming and target specifications.
- **Live Scoring**: Real-time score updates and input.
- **Leaderboard**: Dynamic leaderboard with current rankings and course filtering.
- **Smart Score**: AI-assisted scoring with video replay functionality and improved playback handling.
- **Results Export**: Export tournament results in JSON format.
- **Archer Registration**: Registration form for user details, division selection, and group assignments.
- **Replay Camera**: Captures and uploads video replays triggered by sound detection, optimized for back camera use on iPhones, now showing a live camera feed.
- **Create/Edit Scorecard**: Users can create and edit scorecards linked to tournaments, with photo uploads for thumbnails.
- **Improved Navigation**: Restructured main navigation to include Home, Create Tournament, Create/Edit Scorecard, Live Leaderboard, Results, Replay Camera, and Scorecard.
- **Course Map Upload**: Users can upload course map images when creating tournaments.

# Technology Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS, shadcn/ui
- **Backend**: Atoms Cloud (FastAPI, PostgreSQL, Authentication)
- **SDK**: @metagptx/web-sdk for authentication and API calls

# Usage
1. **Install Dependencies**:
   - For the backend: Run `pip install -r requirements.txt`
   - For the frontend: Run `pnpm install`
   
2. **Build the Project**:
   - For the backend, ensure the database is set up and migrations are applied.
   - For the frontend, run `pnpm run build` to compile the application.

3. **Run the Application**:
   - Start the backend server.
   - Start the frontend server to access the web application.
