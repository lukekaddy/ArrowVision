# Project Summary
**BullsEye Labs** is a mobile-first web application designed to revolutionize archery tournament management through digital scorekeeping. It serves tournament organizers, archers, and scorekeepers with features like live scoring, real-time leaderboards, and instant results. The platform accommodates various tournament formats and advanced scoring options, including mulligans. Recent updates include a new "Replay Camera" feature that allows users to capture and upload video replays of arrow impacts, improving the tournament experience.

# Project Module Description
- **Authentication**: User authentication via Atoms Cloud with role-based access.
- **Tournament Management**: Tools for creating, editing, and managing tournaments, including course naming and target specifications.
- **Live Scoring**: Real-time score updates and input.
- **Leaderboard**: Dynamic leaderboard with current rankings and course filtering.
- **Smart Score**: AI-assisted scoring with video replay functionality.
- **Results Export**: Export tournament results in JSON format.
- **Archer Registration**: Registration form for user details, division selection, and group assignments.
- **Replay Camera**: New functionality for capturing and uploading video replays triggered by sound detection.
- **Create/Edit Scorecard**: Users can create and edit scorecards linked to tournaments, featuring photo uploads for thumbnails.
- **Improved Navigation**: Restructured main navigation to include Home, Create Tournament, Create/Edit Scorecard, Live Leaderboard, Results, and Replay Camera.
- **Course Map Upload**: Functionality allowing users to upload course map images when creating tournaments.

# Technology Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS, shadcn/ui
- **Backend**: Atoms Cloud (FastAPI, PostgreSQL, Authentication)
- **SDK**: @metagptx/web-sdk for authentication and API calls

# Usage
1. **Install Dependencies**:
   - For the backend: Run `pip install -r requirements.txt`
   - For the frontend: Run `pnpm install`
   
2. **Build the Project**:
   - For the backend, ensure the database is set up and migrations are applied.
   - For the frontend, run `pnpm run build` to compile the application.

3. **Run the Application**:
   - Start the backend server.
   - Start the frontend server to access the web application.
