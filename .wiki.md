# Project Summary
**BullsEye Labs** is a mobile-first web application designed to streamline archery tournament management through advanced digital scorekeeping. It caters to tournament organizers, archers, and scorekeepers by providing features such as live scoring, real-time leaderboards, and instant results. The application accommodates various tournament formats and includes tools like a "Replay Camera" for in-depth arrow impact analysis. Recent updates have focused on fixing issues with tournament updates and enhancing user experience.

# Project Module Description
- **Authentication**: Role-based authentication using email/password with bcrypt and local caching.
- **Tournament Management**: Manage tournaments, including course naming and target specifications.
- **Live Scoring**: Real-time score updates and input.
- **Leaderboard**: Dynamic leaderboard with current rankings and course filtering.
- **Smart Score**: AI-assisted scoring and video replay functionality.
- **Results Export**: Export tournament results in JSON format.
- **Archer Registration**: User registration forms with group assignments and duplicate prevention.
- **Replay Camera**: Captures and uploads video replays with improved motion detection.
- **Scorecard Management**: Create and edit scorecards linked to tournaments.
- **Navigation Enhancements**: Improved navigation for better access to functionalities.
- **Course Map Upload**: Users can upload course map images during tournament creation.
- **User Experience**: Pages for archers to view registered tournaments and scorecard history.
- **Profile Management**: Manage accounts and switch roles.
- **Public Tournaments Endpoint**: Fetch public tournaments easily.
- **Debugging Tools**: New endpoint for diagnosing video upload issues.
- **Video Playback Improvements**: Enhanced logging and checks for video playback.
- **Group Management**: Create and manage archer groups with member selection and shooting order.
- **Tournament Deletion**: Users can delete upcoming tournaments after confirmation.
- **Download Replay**: Option for archers to download video replays.
- **Password Reset**: Integrated UI for password reset functionality.
- **My Groups Feature**: Displays all groups the user belongs to, organized by tournament.
- **Scorecards Redesign**: New layout prioritizing active tournaments for quick access.
- **Edit/Delete Buttons for Active Tournaments**: Added functionality to manage active tournaments directly.

# Technology Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS, shadcn/ui
- **Backend**: Atoms Cloud (FastAPI, PostgreSQL, Authentication)
- **SDK**: @metagptx/web-sdk for authentication and API calls

# Usage
1. **Install Dependencies**:
   - For the backend: Run `pip install -r requirements.txt`
   - For the frontend: Run `pnpm install`
   
2. **Build the Project**:
   - For the backend, ensure the database is set up and migrations are applied.
   - For the frontend, run `pnpm run build` to compile the application.

3. **Run the Application**:
   - Start the backend server.
   - Start the frontend server to access the web application.
