# Project Summary
**BullsEye Labs** is a mobile-first web application designed to enhance the management of archery tournaments through advanced digital scorekeeping. It caters to tournament organizers, archers, and scorekeepers, offering features such as live scoring, real-time leaderboards, and instant results. The platform accommodates various tournament formats and integrates innovative tools like a "Replay Camera" for detailed arrow impact analysis. Recent enhancements focus on user experience improvements, including streamlined group management, simplified tournament information access, and a redesigned Scorecard feature that mimics a bowling alley scoreboard for intuitive score tracking.

# Project Module Description
- **Authentication**: Role-based authentication with email/password and phone number verification.
- **Tournament Management**: Tools for creating and managing tournaments, including course naming and target specifications.
- **Live Scoring**: Real-time score updates and input.
- **Leaderboard**: Dynamic rankings with course filtering.
- **Smart Score**: AI-assisted scoring and video replay functionality.
- **Results Export**: Export tournament results in JSON format.
- **Archer Registration**: User registration forms with group assignments and countdown timer for registration.
- **Replay Camera**: Video capture and upload with motion detection.
- **Scorecard Management**: Creation and editing of scorecards linked to tournaments.
- **Navigation Enhancements**: Improved user navigation for better access to functionalities.
- **Course Map Upload**: Upload course map images during tournament creation.
- **User Experience**: Pages for archers to view registered tournaments and scorecard history.
- **Profile Management**: Account management and role switching.
- **Public Tournaments Endpoint**: Easy access to public tournaments.
- **Debugging Tools**: Endpoint for diagnosing video upload issues.
- **Video Playback Improvements**: Enhanced logging for video playback.
- **Group Management**: Create and manage archer groups with member selection.
- **Tournament Deletion**: Users can delete upcoming tournaments.
- **Download Replay**: Option for archers to download video replays.
- **Password Reset**: Integrated UI for password reset.
- **My Groups Feature**: Displays user groups organized by tournament.
- **Scorecards Redesign**: New layout prioritizing active tournaments.
- **Edit/Delete Buttons for Active Tournaments**: Direct management of active tournaments.
- **Sign-out Navigation Fix**: Redirects users to the landing page after signing out.
- **Profile Icon Navigation**: Profile icon now links to the profile page.
- **Group Status Display**: Buttons for archers to find or create groups.
- **Auto-Advance Scoring**: Client-side auto-advance for shooter scoring, ensuring correct shooter highlighting based on scoring status.

# Technology Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS, shadcn/ui
- **Backend**: Atoms Cloud (FastAPI, PostgreSQL, Authentication)
- **SDK**: @metagptx/web-sdk for authentication and API calls

# Usage
1. **Install Dependencies**:
   - For the backend: Run `pip install -r requirements.txt`
   - For the frontend: Run `pnpm install`
   
2. **Build the Project**:
   - Ensure the database is set up and migrations are applied for the backend.
   - For the frontend, run `pnpm run build` to compile the application.

3. **Run the Application**:
   - Start the backend server.
   - Start the frontend server to access the web application.
