# Project Summary
**BullsEye Labs** is a mobile-first web application revolutionizing archery tournament management by digitizing traditional paper scorecards. The platform enhances experiences for tournament organizers, archers, and scorekeepers through features like live scoring, real-time leaderboards, and instant results. It supports various tournament formats and advanced scoring options, including mulligans. Recent updates have improved the scorecard user interface and introduced realistic video replays of arrow impacts, enhancing scoring accuracy and user engagement. A new role-based system allows users to select their roles (Admin or User) upon login, streamlining access to features based on user type.

# Project Module Description
- **Authentication**: User authentication via Atoms Cloud, now includes role-based access.
- **Tournament Management**: Tools for creating, editing, and managing tournaments, including course naming and target specifications.
- **Live Scoring**: Real-time score updates and input.
- **Leaderboard**: Dynamic leaderboard with current rankings and course filtering.
- **Smart Score**: AI-assisted scoring with video replay functionality.
- **Results Export**: Export tournament results in JSON format.
- **Archer Registration**: Registration form for user details and division selection.
- **Role Selection**: New module for users to select their roles post-login.

# Technology Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS, shadcn/ui
- **Backend**: Atoms Cloud (FastAPI, PostgreSQL, Authentication)
- **SDK**: @metagptx/web-sdk for authentication and API calls

# Usage
1. **Install Dependencies**:
   - For the backend: Run `pip install -r requirements.txt`
   - For the frontend: Run `pnpm install`
   
2. **Build the Project**:
   - For the backend, ensure the database is set up and migrations are applied.
   - For the frontend, run `pnpm run build` to compile the application.

3. **Run the Application**:
   - Start the backend server.
   - Start the frontend server to access the web application.
