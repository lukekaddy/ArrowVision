# Project Summary
**BullsEye Labs** is a mobile-first web application that revolutionizes archery tournament management through digital scorekeeping. It caters to tournament organizers, archers, and scorekeepers by offering features like live scoring, real-time leaderboards, and instant results. The platform is versatile, supporting various tournament formats and advanced scoring options, including a "Replay Camera" for capturing video replays of arrow impacts. Recent enhancements include a new endpoint for public tournaments and improved user interfaces for both tournament administrators and archers.

# Project Module Description
- **Authentication**: Custom email/password authentication with role-based access, utilizing bcrypt for password management.
- **Tournament Management**: Tools for creating, editing, and managing tournaments, including course naming and target specifications.
- **Live Scoring**: Real-time score updates and input.
- **Leaderboard**: Dynamic leaderboard displaying current rankings and course filtering.
- **Smart Score**: AI-assisted scoring with video replay functionality.
- **Results Export**: Export tournament results in JSON format.
- **Archer Registration**: Registration forms for user details, division selection, and group assignments exclusive to archers.
- **Replay Camera**: Captures and uploads video replays triggered by sound detection.
- **Create/Edit Scorecard**: Users can create and edit scorecards linked to tournaments.
- **Improved Navigation**: Restructured navigation for better access to Home, Create Tournament, Create/Edit Scorecard, Live Leaderboard, Results, Replay Camera, and Scorecard.
- **Course Map Upload**: Users can upload course map images during tournament creation.
- **Archer Experience**: New pages for archers to view registered tournaments and scorecard history.
- **Profile Management**: Profile page for users to manage accounts and switch roles.
- **Public Tournaments Endpoint**: Added an alias route for fetching public tournaments.

# Technology Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS, shadcn/ui
- **Backend**: Atoms Cloud (FastAPI, PostgreSQL, Authentication)
- **SDK**: @metagptx/web-sdk for authentication and API calls

# Usage
1. **Install Dependencies**:
   - For the backend: Run `pip install -r requirements.txt`
   - For the frontend: Run `pnpm install`
   
2. **Build the Project**:
   - For the backend, ensure the database is set up and migrations are applied.
   - For the frontend, run `pnpm run build` to compile the application.

3. **Run the Application**:
   - Start the backend server.
   - Start the frontend server to access the web application.
